Detailed Description

Class for ioPerhipheral(s). This can be attached to the processor to represent streaming instructions and data to the relevent memories.

Currently, this class writes directly to a given instruction memory or a given data memory. Rationale: Straightforward solution right now to expedite testing of the overall system.

Once further use cases are determined, the class can be further refined to include better abstraction.
